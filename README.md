# Sleep Cycle Calculator

Структура проєкту:

- `index.php` — головна сторінка, вибір мови через `?lang=uk` або `?lang=en`.
- `assets/styles.css` — усі стилі.
- `assets/app.js` — логіка калькуляторів, перемикання мови, темна тема, localStorage.
- `lang/uk.json` — українські тексти, списки, поради.
- `lang/en.json` — англійські тексти, списки, поради.

## Як запустити локально

```bash
php -S localhost:8000
```

Потім відкрити:

```text
http://localhost:8000
```

## Мови

- Українська: `/?lang=uk`
- Англійська: `/?lang=en`

## Що реалізовано

- Розрахунок часу, коли краще лягати.
- Розрахунок часу пробудження.
- Аналіз тривалості сну.
- Калькулятор денного сну.
- План підготовки до сну.
- 15 порад для сну з рандомним показом.
- Темна тема.
- Копіювання результатів.
- Збереження налаштувань у браузері.
- Окремі файли для стилів, логіки та перекладів.
